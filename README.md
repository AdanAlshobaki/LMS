#  LMS Platform — Learning Management System

A Coursera-inspired Learning Management System built with React.js,
featuring course browsing, lesson display, and student progress tracking.

##  Live Demo
[View Live](https://lms-gray-five-83.vercel.app/)

##  Features
-  Browse and explore available courses
-  View detailed lessons within each course
- Track student progress across enrolled courses
-  Fully responsive design — mobile & desktop

##  Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Git & GitHub

##  Installation
git clone https://github.com/AdanAlshobaki/LMS.git
cd LMS
npm install
npm start
